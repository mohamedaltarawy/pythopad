Pythopad.

A free Python source code editor and Notepad replacement for Windows
created by Mohamed Altarawy.

Pythopad allows all the typical
features of a text and code editor
packed into a tiny and fast application

Features

Pythopad brings all the typical features you'd expect from a text/code editor such as a Toolbar, Wide array of view options, Usable accelerator keys (Ctrl+N, Ctrl+O, etc.), Zoom in and zoom out, and Newline ending support.

Syntax Highlighting

Beautiful syntax highligting support for all versions of Python. One of the many features of Pythopad specified for clean, efficient Python development.

Windows exclusive

Pythopad is currently only available on Windows, however, support for Mac and Linux will be brought in the near future.
